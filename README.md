# DockerFarm ERD

DockerFarm ERD 입니다. 

이 프로젝트는 eclipse 기반의 ERD툴인 exERD로 만들어졌습니다. 

프로젝트 수정을 위해서는 아래 Starting 가이드를 참고해주세요. 

## Getting Started

1. 먼저 이클립스와exERD 플러그인을 설치하세요. 

[eclipse](https://www.eclipse.org/)

[exERD](http://ko.exerd.com/#download-section)


2. 현재 프로젝트를 Clone합니다. 

```sh
git clone https://github.com/DockerFarm/DockerFarm-erd.git
```

3. 마지막으로, 이클립스를 실행하고 프로젝트를 Import 하세요. 

## Issue
여기에는 DockerFarm DB설계와 관련된 이슈들만 적어주세요. 


## Versioning
이 프로젝트는 Semantic Versioning을 사용할 예정입니다. 
버전별 ERD는 릴리즈 노트를 통해 Image파일로 배포합니다.
